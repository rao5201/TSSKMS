## 技术栈
- **后端**: Node.js + Express + SQLite
- **移动端**: React Native + Expo
- **管理后台**: React + Ant Design
## 快速开始
### 环境要求
- Node.js 18+
- npm 或 yarn
### 安装步骤
\\\ash
# 1. 安装后端依赖
cd backend
npm install
# 2. 配置环境变量
cp .env.example .env
# 3. 启动后端
npm start
# 4. 安装移动端依赖（新终端）
cd ../frontend
npm install
# 5. 启动移动端
npx expo start
# 6. 安装管理后台依赖（新终端）
cd ../admin-panel
npm install
# 7. 启动管理后台
npm run dev
\\\
### 访问地址
- 移动端: 扫描Expo二维码
- 管理后台: http://localhost:3002
- 后端API: http://localhost:3001
